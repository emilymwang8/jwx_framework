## JWX个人微信公众号框架

#### 安装

运行`/install/install.php` 路径即可初始化安装。

> 当配置好后请删除`install`目录中的`install.php`文件。

#### 目录结构

```
.
├── Common      // 功能函数
├── Config      // 配置文件
├── Controller  // 控制器
├── Core        // 核心配置文件
├── install     // 安装程序
├── Library     // 第三方库
├── Log         // 日志文件
├── Model       // 数据库Model操作文件
├── Public      // 公共资源文件
├── View        // 视图文件
├── index.php
└── README.md

```
